Slasher Manager Account & Data Deletion

Last Updated: July 18, 2026

Developer: Mladen Stojanovic
Contact: mladenstojanovicpo@gmail.com

Slasher Manager does not require an account to play. The data the Game stores for you is limited to local saves on your device and (if enabled) Steam Cloud sync. This page explains how to delete that data.

1. Delete Local Saves

To delete all Slasher Manager data stored locally on your device:

  1. Open the Steam client.
  2. Right-click Slasher Manager in your library → Manage → Browse local files.
  3. Close the Game if it is running.
  4. Delete the folder named app_data (or the equivalent user data folder for Slasher Manager).
  5. Alternatively, uninstall Slasher Manager from your Steam library to remove all local files.

2. Delete Steam Cloud Saves

To delete Slasher Manager data stored in Steam Cloud:

  1. Open the Steam client.
  2. Right-click Slasher Manager in your library → Properties → General.
  3. Uncheck Keep game saves in the Steam Cloud for Slasher Manager.
  4. Click Delete Cloud Data and confirm.

3. Request Manual Deletion

If the steps above do not cover your situation, or if you would like the developer to confirm deletion of any data held outside Steam (for example analytics records associated with your device), email:

mladenstojanovicpo@gmail.com

Please include:

  • Your Steam account name (so we can locate relevant data)
  • Approximate date of the data you want removed
  • A clear statement requesting deletion

We will confirm deletion within 30 days of receiving a verifiable request.

4. Data Not Held by Us

Slasher Manager does not maintain its own user account database. The data described above is held by Steam, by Firebase Analytics (if you opted in), or by your local device storage. Deleting your data through the steps above removes it from those providers as well.
