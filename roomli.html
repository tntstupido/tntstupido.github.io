WHAT IS ROOMLI?

Roomli is a clean, lightweight app for people who share a home. Whether you live with a partner, roommates, or family — Roomli keeps shopping, chores, bills, and expenses in one place so nothing falls through the cracks.

Built with Flutter and Supabase, Roomli syncs in real time across all devices. Add an item to the shopping list and your partner sees it instantly. Assign a chore and everyone knows who's doing what. Track shared bills and expenses so settling up is painless.

FEATURES

Everything you need to run a shared household, nothing you don't.

Shopping Lists

Create shared lists, add items in seconds, check them off together. No more "I thought you bought milk."

Tasks & Chores

Assign and track household tasks. See who's responsible for what, and keep the home running smoothly.

Bills

Track recurring bills and one-time expenses. Never miss a due date or wonder who paid for what.

Expense Tracking

Log shared purchases and see where the money goes. Simple, transparent, no complicated splitting logic.

Real-Time Sync

Changes appear instantly across all devices. Built on Supabase for reliable, fast synchronization.

Household Invites

Create or join a household with invite codes. Easy setup for couples, roommates, or families.

TECH STACK

Roomli is built with modern, reliable tools for a fast and responsive experience.

Flutter

Single codebase for iOS and Android with native performance and smooth animations.

Riverpod

Clean, testable state management that keeps the app responsive and predictable.

Supabase

Real-time database and auth for instant sync across all household members.
